🚀 STEP-BY-STEP DEPLOYMENT ON ANOTHER SYSTEM
============================================

STEP 1: SYSTEM REQUIREMENTS
--------------------------
• Windows 7, 8, 10, or 11 (64-bit recommended)
• Minimum 4GB RAM
• 100MB free disk space
• No Python installation required (EXE is standalone)

STEP 2: COPY THE EXE FILE
-------------------------
• Copy "SchoolManagement_Fixed.exe" to the target system
• Place it in any folder (e.g., C:\SchoolManagement\)
• No additional files needed - EXE is self-contained

STEP 3: RUN THE APPLICATION
--------------------------
• Double-click "SchoolManagement_Fixed.exe"
• Application will launch with console window (this is normal)
• First run may take 10-30 seconds (extracting embedded files)
• Console window shows database creation and file extraction

STEP 4: DATABASE CREATION
-----------------------
• On first run, the application automatically creates:
  - ui2/ folder (contains extracted Python files)
  - ui2/data/ folder
  - ui2/data/school.db (SQLite database)

📁 WHAT GETS CREATED AUTOMATICALLY

🔧 TROUBLESHOOTING
================

❌ COMMON ISSUES AND SOLUTIONS:

1. "ModuleNotFoundError: No module named 'sqlite3'"
   • ✅ FIXED: Use SchoolManagement_Fixed.exe (includes sqlite3)
   • If still occurs: Install Microsoft Visual C++ Redistributable

2. "Application won't start"
   • Solution: Run as Administrator
   • Check if Windows Defender blocked it
   • Try running from Command Prompt

3. "Access denied"
   • Solution: Move EXE to a different folder
   • Avoid Program Files folder
   • Create new folder like C:\SchoolApp\

4. "Database errors"
   • Solution: Delete ui2/data/school.db and restart
   • Application will recreate database

5. "Slow startup"
   • Solution: Normal on first run, subsequent runs are faster
   • Console window shows progress

6. "Console window appears"
   • This is NORMAL - shows database operations
   • Do not close the console window
   • Main GUI window will appear after initialization

7. "Missing DLL errors"
   • Solution: Install Microsoft Visual C++ Redistributable
   • Download from Microsoft website (free)
