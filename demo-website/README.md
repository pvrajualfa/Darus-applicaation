# TechLearn Demo Website

A modern, responsive demo website built with HTML, CSS, and JavaScript for practicing web development skills.

## 🚀 Features

### **Core Functionality**
- **Responsive Navigation**: Mobile-friendly hamburger menu with smooth animations
- **Hero Section**: Eye-catching landing area with call-to-action buttons
- **About Section**: Feature cards showcasing key benefits
- **Courses Section**: Interactive course cards with enrollment functionality
- **Contact Section**: Working contact form with validation
- **Footer**: Complete footer with social links and site information

### **Interactive Features**
- **Smooth Scrolling**: Navigation links scroll smoothly to sections
- **Mobile Menu**: Hamburger menu for mobile devices
- **Scroll Animations**: Elements animate when scrolling into view
- **Parallax Effects**: Hero section has subtle parallax scrolling
- **Hover Effects**: Interactive hover states on all cards and buttons
- **Modal System**: Course enrollment modal with form validation
- **Notifications**: Toast-style notifications for user feedback
- **Form Validation**: Client-side validation for all forms
- **Typing Effect**: Hero title has a typing animation

### **Design Features**
- **Modern UI**: Clean, professional design with gradient accents
- **CSS Variables**: Consistent color scheme and easy customization
- **Grid Layouts**: CSS Grid for responsive layouts
- **Flexbox**: Flexible layouts for components
- **Animations**: Smooth transitions and micro-interactions
- **Typography**: Clean, readable font hierarchy
- **Color System**: Professional color palette with CSS variables

## 🛠 Technologies Used

- **HTML5**: Semantic markup with proper structure
- **CSS3**: Modern CSS with Grid, Flexbox, and animations
- **Vanilla JavaScript**: No frameworks, pure JavaScript
- **Font Awesome**: Icon library for visual elements
- **Responsive Design**: Mobile-first approach

## 🎨 Customization

### **Colors**
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #1e40af;
    --accent-color: #3b82f6;
    /* ... more variables */
}
```

### **Fonts**
Change fonts in the CSS:
```css
body {
    font-family: 'Your Font', sans-serif;
}
```

### **Animations**
Modify animation speeds and effects in the CSS and JavaScript files.
